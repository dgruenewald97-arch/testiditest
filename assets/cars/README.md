# Fahrzeug-Bilder (Platzhalter)

Diese SVG-Dateien sind **freigestellte Platzhalter-Autos**. So tauschst du sie
gegen die echten Mitsubishi-Freisteller:

## Variante A – einfach gleiche Dateinamen nutzen
Lege das echte Bild mit **demselben Namen** hier ab, z. B.:

- `outlander.svg`  →  `outlander.png`
- `outlander-black.svg`
- `asx.svg`
- `grandis.svg`
- `eclipse-cross.svg`
- `l200.svg` (Pick-up)

Dann in `index.html` im `OFFERS`-Array nur die Endung im `img`-Pfad anpassen
(`.svg` → `.png`).

## Variante B – Pfad direkt ändern
In `index.html` im `OFFERS`-Array beim jeweiligen Modell `img:"…"` auf den
neuen Dateipfad zeigen lassen.

## Empfehlung für echte Bilder
- **Freigestellte PNGs** mit transparentem Hintergrund (kein weißer Kasten)
- Seitenansicht, gleiche Blickrichtung wie die Platzhalter (Front rechts)
- Breite ca. 1600 px für scharfe Darstellung
